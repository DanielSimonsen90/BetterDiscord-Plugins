# [PronounsPageLinks v1.0.0](https://github.com/DanielSimonsen90/BetterDiscord-Plugins/dist/bd/PronounsPageLinks)
by danhosaur

## What does it do?
Converts pronouns.page "links" in UserPronouns to actual clickable links.
